# Intelligent AI-Driven Marketing System & Executive Analytics Command Center

An end-to-end intelligent data analytics system designed to optimize retail marketing operations for Chief Marketing Officers (CMOs). The system ingests raw consumer behavioral data, filters noise using an automated Python data processing pipeline, performs Natural Language Processing (NLP) sentiment analysis, clusters customers via unsupervised machine learning, and calculates predictive churn metrics. All intelligence is funneled directly into a multi-page, cloud-hosted Power BI executive dashboard equipped with advanced custom tooltips, interactive bookmark selection mechanics, an integrated standalone AI Q&A natural language query block, and a pinned key metric executive monitoring panel.

## 🛠 Tools & Libraries Used

### Data Processing & Machine Learning (Python)
*   **pandas / NumPy**: For currency sanitization, handling missing values, and executing data cleaning pipelines.
*   **scikit-learn**: For engineering unsupervised K-Means clustering frameworks and computing classification metrics for predictive customer churn models.
*   **spaCy / NLTK**: For unstructured text processing, tokenization, lemmatization, and natural language sentiment polarity scoring.
*   **pydub**: For multi-media preprocessing pipelines within adjacent data models.

### Business Intelligence & Visualization
*   **Power BI Web Authoring / Desktop**: For multi-page analytical dashboard creation and interactive report development.
*   **Power BI Service (Cloud Workspace)**: For pinning standalone executive dashboards and deploying cloud-synchronized semantic data modules.
